# Markdown 格式模板与规范

## 输出模板

```markdown
# [页面标题]

> 来源： [URL]
> 日期： YYYY年MM月DD日

---

[正文内容]

---

*内容由 AI 提取，不能完全保障原始内容的完全准确性*
```

## 标题层级映射

| HTML 标签 | Markdown 语法 |
|-----------|-------------|
| `<h1>` | `# ` |
| `<h2>` | `## ` |
| `<h3>` | `### ` |
| `<h4>` | `#### ` |
| `<h5>` | `##### ` |
| `<h6>` | `###### ` |

## 语义标签转换

| HTML 标签 | 转换为 |
|-----------|--------|
| `<strong>`, `<b>` | `**text**` |
| `<em>`, `<i>` | `*text*` |
| `<code>` | `` `code` `` |
| `<pre><code>` | ` ```lang\ncode\n``` ` |
| `<a href="...">` | `[text](url)` |
| `<blockquote>` | `> text` |
| `<ul><li>` | `- item` |
| `<ol><li>` | `1. item` |
| `<hr>` | `---` |
| `<img src="...">` | `![alt](src)` |

## 应移除的标签

- `<script>` — 全部移除
- `<style>` — 全部移除
- `style="..."` 属性 — 移除
- `class="..."` 属性 — 移除
- `onclick="..."` 等事件属性 — 移除

## 保留的属性

- `<a href="...">` 的 `href`
- `<img src="...">` 的 `src`
- `<pre>` 的 `class`（提取语言标记）
